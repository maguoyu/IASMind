## 第3部分 驾驶操作与行驶规范

### 3.1 驾驶前准备
【关键词】启动前检查、安全带、车门、控制检查、预检查单

**驾驶位置操作规范**
✓ 确定系好安全带并关上车门
✓ 启动前进行控制杆、踏板检查
✓ 检查后视镜视线良好
✗ 禁止在驾驶室外启动引擎

**启动前检查清单**
- [ ] 设备、后视镜视线是否良好
- [ ] 操控杆及其他部件状况
- [ ] 信号传输功能正常
- [ ] 紧急停机可用
- [ ] 罩子完全闭合并锁定
- [ ] 检查泄漏情况
- [ ] 轮胎损伤检查
- [ ] 相关安全部件完好
- [ ] 灭火器有效期检查
- [ ] 所有指示灯正常

### 3.2 行驶操作规范
【关键词】行驶速度、转向、制动、路况适应、交通指示

**速度和路况调整**
- 调节车速和车辆方向，符合驾驶区域要求
- 考虑因素: 可见性、斜坡、拖动负载、十字路口、路宽和高度
- 适应交通情况: 行人、飞机等周边环境

**可见性处理**
- 路况可见性低时，需要专业人员领路
- 使用反光镜，避免盲区操作
- 特别注意倒车时的安全

**指示信号使用**
✓ 所有改变行驶方向时打开方向指示灯
✓ 打开信号灯让其他人员注意
✓ 使用喇叭提示可能遭遇的危险

### 3.3 停车和制动系统
【关键词】停车制动、驻车、制动系统、制动辅助、停放位置

**主制动与电机制动**
- **主制动**: 踩下刹车踏板，作用于前轮
- **电机制动**: 当加速踏板释放时自动作用

**驻车制动（停车）**
- 拉驻车制动器（位于驾驶位左侧）
- 排挡杆置于空档
- 驻车制动指示灯应闪烁（频率500ms）

**停车位置要求（服务维护）**
✓ 确保紧急停机开关未被按下
✓ 运用电池切断建立电路
✓ 系上安全带
✓ 排挡杆处于空档
✓ 客梯在最低位置（除对接和收回时）
✓ 支腿已回收
✓ 安全栅处于正确位置

**油泵阀门维护**
⚠️ 打开阀门替代泵的作用但不包括排油
⚠️ 油泵电机工作时需保持阀门开通
✗ 阀门关闭会导致液压泵损坏

## 第4部分 专项操作规范

### 4.1 与飞机对接操作
【关键词】对接流程、舱门对齐、平台调整、安全距离、对接程序

**对接前准备**
- 以低速（电机空转）前进至飞机舱门对应位置
- 与机身保持垂直
- 客梯升起时保持电机空转
- 在飞机前停下，挂空档

**对接步骤（顺序操作）**
1. 向左操纵阶梯控制操纵杆，调整俯仰油缸使平台水平
2. 向前操纵阶梯控制操纵杆，升起客梯至服务舱门
3. 对准十字板上相应标记
4. 稍许超出电磁阀固定块以固定客梯
5. 电磁阀启动时释放操控杆

**平台倾斜调整**

【关键词】B747、舱门对接、特殊步骤、通风片距离、特殊程序

【关键词】存放环保、防腐蚀、定期检查、投入使用前准备、存放计划

**存放环境要求**
✓ 遮蔽处存放
✗ 避免阳光暴晒
✗ 避免风沙吹蚀
✗ 避免雨淋
✗ 避免高盐度环境

- **铅酸电池**: 检查电解液位，如有必要加满
- **定期检查**: 每3个月检查电气件连接，防止损坏或腐蚀

- 清理设备外部
- 未油漆表面金属部件需做防腐蚀涂层并涂抹润滑脂
- 每3个月检查腐蚀情况，若产生腐蚀立即清理并更换保护层
- 用楔垫抬起车架，使车轮远离地面

**液压部件维护**
- 关闭前收回油缸
- 对不能收回的油缸（转向油缸）围绕轴心涂抹润滑脂
- 彻底清理油箱外部、液压元件和管路
- 修复液压油泄漏
- 封堵液压油箱出口，确保液压油箱密封
- 使用润滑脂将液压阀固定装置做防锈处理
- 所有液压元件、管路、法兰及连接器上涂抹防锈液
- 每年定期排干并更换油箱、液压泵、机舱中的液压油
- 每3个月检查液压元件上是否有锈蚀，特别是未收回的油缸及液压连接件

**投入使用前准备**
- 打开油箱盖并更换油箱口滤芯
- 去除液压管路上的油脂条
- 排空、清理并重新注满新的干净液压油
- 排空并注满液压泵及发动机组
- 更换回油滤芯
- 前50小时使用后，更换新的回油滤芯

**存放超过6个月**

### 8.1 车身结构

【关键词】车身、结构、刚性梁、转向系统、制动系统、液压油缸、防撞挡块

**基本结构**

车身结构由刚性梁和钢横柱焊接而成。液压助力转向系统转动前轮。四轮都有液压制动（前轮盘刹，后轮鼓刹，独立前后回路），由拉线控制的后轮驻车制动。

**稳定性设计**

四支垂直液压油缸用支撑板和球形支座固定在地面，保证整车的稳定性。所有操作（驾驶和控制）均在前驾驶室内完成。防撞挡块保护车辆前部和侧部。

### 8.2 客梯和平台

【关键词】客梯、平台、登梯、伸缩台阶、安全栅、雨篷、照明灯

**台阶系统**

伸缩式台阶由液压缸定位，提供按阶平台调整。固定台阶被铆在框架上，因此可以通过液压缸倾斜实现精确的门高最后调整。和固定台阶对应的伸缩台阶（可动）由电磁铁安全固定，只有在降下台阶时才能解除锁止。

**结构特性**

- 整体阶梯呈┗┛形，以保证高强度和刚性
- 侧体用于支撑扶手和台阶照明
- 自支撑型铝合金台阶有防滑层
- 内部台阶鼻保证脚部安全
- 台阶宽度足够两人通过

**平台功能**

- 装有可吸收震动的EPDM橡胶保险杠的平台前端，可以绕轴转动，在±10°的范围内自动对齐
- 装有吸震保险杠的侧门和滑轨同样可以滑动一个足够的距离，以便客机打开舱门
- 主平台和中间平台的地板是由自承重的防滑铝合金制成
- 客梯侧轨上的灯为台阶照明

### 8.4 蓄电池系统

【关键词】蓄电池、锂电池、电源、PDU控制盒、灭火系统、电池管理

**电池系统概述**

大容量电池系统确保为整车的运行以及电气元件提供稳定、高效的能量来源。蓄电池使用寿命长，性能稳定安全可靠。

**锂电池配置**

布局在车架中间两个或三个(选装)277AH的锂电池，具有以下特点：
- 能量密度高
- 重量轻
- 自放电率低
- 充电效率快
- 成为新能源时代的首选产品

**电源管理**

电源分配控制盒（PDU）是为安装的电气设备提供电力分配而设计的产品，能为不同的电源环境提供适合的电源分配解决方案。PDU的应用可使电源分配更加整齐、可靠、安全、专业和美观，并使得电源的维护更加便利和可靠。

**灭火保护系统**

灭火系统为每个标准电池包配置一个灭火瓶，用于火灾探测（紫外、红外、烟雾）。灭火剂为NY5112全氟己酮，具有绝佳的环保性能和低毒性特征，灭火高效、环保洁净、使用安全、对电子精密设备无损伤。
